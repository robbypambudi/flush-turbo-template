# Flush Turbo Template

Do you want to build a modern web application with the latest technologies? Turbo is a new way to make your web app faster and more interactive. It's a set of tools that help you build a fast, modern web application without having to write a lot of code.

## Why Flush

Flush is a modern web application template that uses monorepo architecture to manage multiple applications in a single repository. It's built with Turbo, React, and TypeScript. It's designed to be fast, easy to use, and flexible. All documentation is written in Markdown and rendered with MDX. It's a great way to understand what Flush can do for you.

## Features

### Frontend

- **Turbo**: A set of tools that help you build a fast, modern web application without having to write a lot of code.
- **Next.js**: A JavaScript framework that lets you build server-side rendering and static web applications using React.
- **React**: A JavaScript library for building user interfaces.
- **TypeScript**: A typed superset of JavaScript that compiles to plain JavaScript.

### Backend

- **Golang**: A statically typed, compiled programming language designed at Google.
- **PostgreSQL**: A powerful, open-source object-relational database system.
- **Docker**: A platform for developing, shipping, and running applications in containers.
